%b, %B Umumiy true yoki false
  • String.format("%b", true) -> true
  • String.format("%B", false) -> FALSE
  • String.format("%b", null) -> false
%h, %H Umumiy Ob'ekt hash kodining Hex ko'rinishi
  • String.format("%h", new Student()) -> 4554617c
  • String.format("%H", new Student()) -> 4554617C
%s, %S Umumiy String (matn,tekst)lar uchun ishlatiladi
  • String.format("%s", "Alish") -> Alish
  • String.format("%S", "Alish") -> ALISH
%d char (ishora,belgi) lar char(ishora, matn)lar uchun ishlatiladi
  • String.format("%c", 'a') -> a
  • String.format("%C", 'a') -> A
%d integer char(ishora, matn)lar uchun ishlatiladi
  • String.format("%d", 22) -> 22
%o integer Octal sonlar (8 sanoq sistemasidagi sonlar) uchun ishlatiladi.
  • String.format("%o", 10) -> 8
%x, %X integer Hexadecimal sonlar (16 sanoq sistemasidagi sonlar) uchun ishlatiladi.
  • String.format("%x", 13) -> d
  • String.format("%X", 13) -> D
%e, %E floating point e darajasidagi kasr sonlar uchun (ilmiy soxada ishlatiladi.)
  • String.format("%e", 3.05f) -> 3,050000e+00
  • String.format("%E", 4.05f) -> 4,050000E+00
%f floating point Kasr sonlar uchun.
  • String.format("%f", 33.05) -> 33,050000
%g,%G floating point Kasr sonlar uchun, kasr qismini yaxlitlaydi.
  • String.format("%g", 33.1234567) -> 33.1235
  • String.format("%G", 33.490001f) -> 33.4900
  • String.format("%G", 3,59111111) -> 3,59111
%a,%A floating point Hexadecimal kasr sonlar uchun. (0x1.4333333333333p3)
  • String.format("%a", 13.5 ) -> 0x1.bp3
  • String.format("%A", 1.8 ) -> 0X1.CCCCCCCCCCCCDP0
%t,%T date/time Sana (data) va vaqt(time) uchun ishlatiladi. Bu oddiy prefiks (oldida turadigan ishora). UNdan keyin turadigan harflar bor. Ularni bir pastga alohida ko'ramiz. %% percent % (foizni) chiqarish uchun ishlatiladi.
  • String.format("%S") -> %
%n line separator Qatorlarni ajratadi. Hozirgi qatorni bo'lib yangi qatordan boshlaydi.
  • Xuddi System.getProperty("line.separator") kabi ishlaydi.
